# Aqua Acrobatics Unofficial

Modern swimming, crawling and elytra flight for Minecraft 1.12.2 - Unofficial Fork

## Features
- ✅ Modern swimming mechanics (like 1.13+)
- ✅ Crawling to fit through 1-block gaps
- ✅ Elytra takeoff from ground (1.15+ style)
- ✅ New water rendering and fog colors
- ✅ Bubble columns (Magma/Soul Sand)
- ✅ Floating items in water
- ✅ Compatible with many mods
